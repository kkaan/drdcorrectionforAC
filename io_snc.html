Rearranges the detector data from acl file into the one displayed in SNC Patient.

Parameters: acl_detectors (DataFrame): The measurement data arranged in acl formal.

Returns: arrays (ndarray): A 3D numpy array created from the differential dose data.

io_snc.diode_numbers_in_snc_array()[source]

Reorganizes the detectors numbers in an acl measurement file into the planar array that is displayed in SNC Patient software.

Parameters: acl_file (str): The path to the acl measurement file.

Returns: array (ndarray): A 2D numpy array representing the reorganized detectors.
